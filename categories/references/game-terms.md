# P2 游戏内名词解释

## 简答
> AC=卫星战场、GVG=火车竞赛、KVK=银河战争、KVK2=异星争霸、ZVZ=最强猴子跨服

---

## 活动名词对照表

| 缩写 | 中文名称 | 说明 |
|------|----------|------|
| **AC** | 卫星战场 | Alliance Championship |
| **GVG** | 火车竞赛 | Guild vs Guild |
| **KVK** | 银河战争 | Kingdom vs Kingdom |
| **KVK2** | 异星争霸 | KVK 第二阶段/新版本 |
| **ZVZ** | 最强猴子跨服 | Zone vs Zone |
