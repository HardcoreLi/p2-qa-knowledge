# 平台 SDK 涉及功能清单

## 简答
> P2 平台 SDK 涉及 12 大功能模块：GP/iOS 支付、推送、翻译、聊天、auth/fb/gp/ios 登录、上传头像头像框、邮件。测试时需覆盖各渠道的 SDK 差异。

---

## SDK 功能模块一览

| 序号 | 功能模块 | 说明 |
|------|----------|------|
| 1 | **GP 支付** | Google Play 商店支付 |
| 2 | **iOS 支付** | App Store 支付 |
| 3 | **推送** | 个人推送、全体推送 |
| 4 | **翻译** | 聊天翻译、联盟公告翻译、地图标记翻译 |
| 5 | **聊天** | 全服/氏族/帮派/私聊频道 |
| 6 | **Auth 账号登录** | 游客账号、本地账号 |
| 7 | **FB 登录** | Facebook 账号登录 |
| 8 | **GP 登录** | Google Play 账号登录 |
| 9 | **iOS 登录** | Apple ID 登录 |
| 10 | **上传用户头像** | 自定义头像上传 |
| 11 | **上传头像框** | 自定义头像框上传 |
| 12 | **邮件** | 收发邮件、战报 |

### Discord 特殊规则

| 场景 | 显示规则 |
|------|----------|
| 登录界面 | 所有包**不显示** Discord 账号登录 |
| 绑定弹窗（国际服） | **显示** Discord 账号绑定 |
| 绑定弹窗（国服） | **不显示** Discord 账号绑定 |

### 账号注销功能

以下渠道**必须有**账号注销入口：
- iOS 世界服
- iOS 越南
- iOS 国服
- 国服华为

其他渠道**无需**账号注销功能。

## 常见 SDK 问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 支付失败（货币单位非美元） | 支付账号拉取错误 | GP 商店确认账号后重启客户端 |
| 拉不起支付弹窗 | 没有登录支付账号 | 设备设置中重新登录支付账号 |
| GP 游戏账号登录失败 | 设备无原生谷歌框架 | 换有谷歌框架的设备 |
| 推送收不到 | Win 包不支持 / 安卓无谷歌框架 | 确认设备环境 |
| 第三方账号登录失败 | 账号被封 | 切换其他有效账号 |
