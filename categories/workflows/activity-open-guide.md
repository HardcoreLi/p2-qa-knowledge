# 活动开启流程指南

## 简答
> 活动开启分为：后台开启、GM 指令开启、条件触发开启。不同活动有不同的前置条件和流程，KVK/联盟战场/AC联赛等跨服活动需要确保 cross/battle/subcross 服正常启动。

---

## 一网打尽

### 活动开启方式
1. KVK 内通过天下大势直接解锁
2. 原服通过后台开启

### 开启挑战条件
- 帮派内有有效的帮派建筑（旗帜或帮派总部）
- 帮派内人员达到 **15 人以上**
- 有联盟币（`inituasset`）
- 帮派内 **R4、R5** 才能开启

### 挑战玩法
- 击败刷新出的野怪得积分：`kvkgvescore xx` 直接加积分
- 击败完所有野怪且目标配送站血量不为 0：`kvkgvesuccess n`（n 为想要击败的波次）

## KVK 开启

### 非常规开启（GM 指令）
```
movekvk xxx（xxx 为 kvk 服 id）
```

**注意事项**：
1. 要进的 kvk 服与当前所在的 game 服在同一台物理机下
2. 对应的 kvk 服是正常开启的
3. 使用指令前，确保账号有帮派、氏族，**开服 15 天**

### 常规开启流程
1. 服务器清数据
2. 后台部署活动：预告时间 24h、活动开启时间为当前时间第二天 00:00:00 开始，7d 后结束
3. 调服务器时间到当前时间的前一天 23:59:00，自然度过 1 分钟跨过 0 点开启预告
4. 预告开启成功后等待 1 分钟左右，调服务器时间到当前时间的 23:59:00，自然度过一分钟跨过 0 点开启预热活动

**关键点**：
- 活动必须要用预告期（预告期 1 天，活动持续 7 天 168h）
- 测试匹配至少保证有 **7 个部落**参与
- cross 改成对应的 kvk 服（加 `--kvkid` 配置）
- 需要调时间保证注册时间超过 15 天

### 使未报名玩家变成报名玩家
1. 新号加入报名成功的联盟
2. 进编译工具（https://c.runoob.com/front-end/693/）查出新号序号：
```lua
p = PlayerMgr:GetPlayer(玩家ID)
p:UnlockKvkSignHorde()
p.Stat.RegisterTs = 1000000
```
3. 用 postman 热更：
```json
{
  "server_ids": [要热更的服务器id],
  "cmd": "pluabase64",
  "args": ["序号"]
}
```
4. 重启客户端

## 联盟战场开启

### 非常规开启（GM 指令）
1. 两个玩家创建两个联盟
2. 其中一个联盟的玩家使用：`ubattleapply xxx`（xxx 为另外一个联盟的简称）
3. 所有两个联盟里的玩家使用：`enterub` 即可进入联盟战场

**注意**：确保对应的 battle、cross 服均正常开启

### 常规开启流程
1. 后台部署活动
2. 报名后调时间到匹配当日北京时间 8:01:30，正常过渡至 8:02:15
3. 匹配结束需要自然跨过 UTC 0 点
4. 调整到开战当日 20:54:00，正常过渡到 20:56
5. 调整到开战当日 21:53:30，进入界面等待进入战场

## AC 联赛

### 常规开启流程
1. 后台需要开启 **AC 联赛主活动**和**参赛详情**两个活动
2. 报名后调时间到匹配当日 UTC 0 点开始匹配
3. 自然跨过匹配期 0:02，调时间到 0:11 自然跨过 12 分可以出匹配结果
4. 匹配结束需要自然跨过 UTC 0 点

---

## 太空争霸玩法

### 预告
1. 正常清服开服，天下大势阶段调至太空争霸前一个阶段
2. 主城升至 **6 级以上**

### 活动开启
预告结束即天下大势阶段到达太空争霸阶段
> 调时间用 `nextday 1`

### 押运玩法
1. 活动开启
2. 帮派占领任意一个太空城建筑，占领成功后服务器调 4h 时间到达保护期
3. 太空城建筑旁边立即出现火车

> ⚠️ **一定记得修仓库，不然押运界面会炸**

### GVE 玩法
火车到巨猿位置正确操作步骤：
1. 使用 `traingocenter` 指令使火车开始行驶，调时间 15min，火车到达一个站点会停下
2. 同样操作四次，火车走到巨猿位置（距离发射台 80% 路程）则立即开启巨猿玩法

## 攻城略地

### 活动开启
1. game 服、cross 服、subcross 服、battle 分支正确，都在启动状态，door 在启动状态
2. 后台或者 `faddactv` 开启活动都行，跨服需要后台开
3. 活动开启后第一阶段处于报名期，至少两个联盟报名
4. 报名成功后，调物理机时间自然过渡到下一阶段
5. 第二阶段为匹配期，等三分钟，确保看见报名的两个联盟信息都出现
6. 第三阶段为战斗期，匹配成功的玩家点击进入战场按钮即可进入战场

> **注意**：匹配期只有 12 小时，不要调跨天了

---

## 巅峰领土战

### 开启流程（新版）
1. 保证 game 服、cross 服、kvk43 服都在启动状态，分支相同
2. 在 game1 服建 1 个联盟并加入氏族，在 game2 服建 1 个联盟并加入另一个氏族（至少需要 2 个同 group_id 的 game 服）
3. 在后台开启活动，预告期至少 24 小时，活动开两天
4. 自然过渡到活动预告期 0 点，之后物理机时间往后调 29 分钟，自然过渡 0:30
5. 重启客户端，查看活动界面，每个阵营显示了氏族信息则匹配成功

### 战斗流程
6. 物理机调时间到预告期当天 23:59，自然过度到下一天，可进入战场
7. 开战第一天，物理机调时间到 5:59，自然过渡到 6:00
8. 物理机调时间到 11:59，自然过渡到 12:00，可开始战斗
9. 物理机调时间到 17:59，自然过渡到 18:00
10. 物理机调时间到 21:59，自然过渡到 22:00
11. 物理机调时间到 23:28，自然过渡到 23:30，死兵进入特殊医院

### 清除匹配信息
```json
{
    "server_ids": [x1,x2],
    "cmd": "cleanmatch",
    "args": [],
    "all_server": true
}
```

## 跨服围剿 BOSS

### 同个 subcross
1. 确保原服、subcross、cross、battle 服都在启动状态
2. 确保两个原服的 group 一致
3. 修改 1013 表内的开启跨服围剿 boss 的难度等级
4. 两个服达到了解锁难度后，需要同时跨过**周二的 UTC 零点**

> **注意**：刚跨过周二 UTC 零点后一分钟内无法组队是正常现象

---

## 新跨服战场（王者之地）

### 首场流程
1. 确保 game、cross、battle 服都在启动状态
2. 活动开启，持续时间必须是 **28 天**，否则匹配不成功
3. 进行报名（单独报名或组队报名）
4. 报名成功后，修改时间到当天 21:59:30，自然过渡
5. 进入匹配期后，修改时间到跨天 01:59:30，自然过渡
6. 进入开赛提醒阶段，修改时间到当天 11:59:30，自然过渡

> GM 指令：`faddactv 2111xxxx 0 672h`

## 合服活动

### 开启
1. 确保 door、game、cross、sub_cross 等服均正常启动
2. ark 后台开启，选择合服前活动 id：**21112110**
3. 是否跨服选择跨服，将要合服的服务器 id 填在一个分组内
4. 自定义参数栏按格式填写：`A,B|F,G`（代表 AB 一组，FG 一组）

---

## 常见坑点

- **跨服活动**：必须确保 cross/battle/subcross/door 服都在启动状态且分支一致
- **时间调整**：很多活动需要用物理机调时间，单服时间可能不生效
- **匹配时间窗口**：部分活动匹配期很短，注意不要调跨天
- **预告期**：KVK 等活动必须有预告期才能正常开启
- **注册时间**：KVK 需要账号注册时间超过 15 天
