# 合服需求确认流程

## 简答
> 合服前一周需完成两项确认：1）在 dev 群 @ 服务器同学更新版本发布表的合服处理信息；2）在节日活动合服处理群跟运营林康确认合服处理需求。

---

## 时间节点

**合服前一周**定期确认合服时间，启动需求确认流程。

---

## 确认事项

### 一、版本发布表更新

**操作位置**：dev 群

**@ 对象**：
- Allen 张涵
- Cui-崔紫航
- Muzig 李钢
- 徐琮淦
- 周垚旭

**消息模板**：
```
@Allen 张涵 @Cui-崔紫航 @Muzig 李钢 @徐琮淦 @周垚旭 
麻烦将XX版本有合服处理的功能标记下
https://docs.google.com/spreadsheets/d/1pMcoO2RmNuvfJV1r0er_rEMdPET5sEME_jgkX-DrxaI/edit?gid=170013521#gid=170013521
```

**目的**：让服务器同学在版本发布表中标记当前版本涉及合服处理的功能点。

---

### 二、节日活动合服处理确认

**操作位置**：节日活动合服处理群

**确认对象**：运营 林康

**确认内容**：
- 合服处理需求
- 一般会提供处理表格
