# 新服测试流程

## 简答
> 新服上线前的测试验证流程，确保新服务器配置正确、功能正常。

## 测试时机
- 新服开放前 1-2 天
- 运营提供新服信息后

## 测试准备

### 1. 获取新服信息
- 服务器 ID
- 开服时间
- 服务器配置

### 2. 准备测试账号
- 新建测试账号
- 或使用专用新服测试账号

### 3. 确认测试环境
- 客户端版本正确
- 能够连接到新服

## 测试内容

### 1. 基础功能验证
- [ ] 创建角色正常
- [ ] 新手引导完整
- [ ] 基础操作正常（建造、升级等）

### 2. 活动验证
- [ ] 开服活动显示正确
- [ ] 活动奖励可领取
- [ ] 活动时间正确

### 3. 充值验证
- [ ] 充值入口正常
- [ ] 首充奖励正确
- [ ] 商城显示正常

### 4. 服务器配置验证
- [ ] 服务器名称正确
- [ ] 服务器时区正确
- [ ] 跨服功能正常

### 5. 数据验证
- [ ] 排行榜正常
- [ ] 联盟功能正常
- [ ] 邮件系统正常

## 测试报告

### 报告内容
1. 测试时间
2. 测试服务器
3. 测试结果（通过/不通过）
4. 发现的问题
5. 建议

### 报告模板
```
【新服测试报告】
服务器：xxx
测试时间：xxxx-xx-xx
测试人：xxx

一、测试结果：通过/不通过

二、测试详情：
1. 基础功能：✓
2. 活动验证：✓
3. 充值验证：✓
4. 配置验证：✓

三、发现问题：
- 问题1：xxx
- 问题2：xxx

四、建议：
xxx
```

## 常见坑点
- **时区问题**：注意服务器时区与活动时间
- **配置遗漏**：检查所有开服相关配置
- **缓存影响**：清除缓存后测试
