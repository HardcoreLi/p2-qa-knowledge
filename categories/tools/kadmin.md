# Kadmin 服务器管理工具

## 简答
> Kadmin 是服务器管理后台，用于部署服务器、构建镜像、修改配置、清服等操作。访问地址：http://kadmin.tap4fun.com

## 登录方式
- 使用公司邮箱登录
- 权限不足时联系运维开通

## 核心功能

### 1. 部署服务器
1. 进入 Kadmin 首页，搜索目标服务器
2. 点击服务器进入详情页
3. 点击「部署」按钮
4. 选择部署参数（分支、配置等）
5. 确认后等待部署完成

### 2. 镜像构建
- 用于构建新的服务器镜像
- 一般在代码更新后需要重新构建

### 3. 配置修改
1. 进入服务器详情页
2. 点击「配置」或「编辑」
3. 修改 command 参数或其他配置
4. 保存后需要重新部署生效

### 4. 清服操作
1. 进入服务器详情页
2. 点击「清服」按钮
3. 确认清服（数据会被清空）
4. 清服后需要重新部署

## 常见坑点
- **部署后不生效**：检查是否选择了正确的分支和配置
- **清服失败**：确认服务器已停止运行
- **配置修改不生效**：修改配置后必须重新部署

## 常用服务器命名规则
- `p2-beta-xxx`：测试环境服务器
- `p2-gold-xxx`：线上环境服务器
